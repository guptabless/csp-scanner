# csp-scanner


This is a lightweight tool that scans for the whitelisted URL in the CSP header.

## Requirment:


### packages 

- requests
- bcolors
- sys
- argparse

### python > 3.x 

## usage: 

csp.py  -u < Valid URL with http:// or https:// >

OPTIONS: 

```
-u            valid URL with http:// or https://
  		
```
